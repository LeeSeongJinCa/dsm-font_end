안녕하세요? 저는 1학년 2반 이성진입니다.

제가 DMS에 지원한 이유는 총 3가지입니다.

웹 프론터의 기본

저는 웹 프론트를 시작한 지 약 4개월밖에 되지 않은 웹 초보자입니다. DMS에 들어가 다양한 웹 프론터의 기술을 배우고 실력을 증진하고 싶습니다. 또, 선배들이 직접 짠 코드를 리뷰하면서 다양한 기술의 쓰임을 알고싶습니다.

여러 분야와 협업

DMS는 1학년부터 3학년까지 전체 학년이 모여서 진행하는 팀 단위 프로젝트입니다. 그만큼 다양한 분야의 사람들과 함께할 기회가 많다고 생각합니다. 그렇기에 DMS에 들어가 일찍이 다양한 사람들과 협업을 해보고 싶습니다.

웹 프론트로서 큰 성장의 기회

사용자에게 직접적으로 웹 사이트를 제공하는 것은 웹 프론터의 꿈이라고 생각합니다. 이런 꿈을 고등학교에서 실현할 수 있는 기회는 DMS라고 생각합니다. 게다가, DMS는 대마고를 학생을 위한 프로그램을 만들기 때문에 직접적이고 빠른 피드백을 받을 수 있습니다.

저를 뽑아주신다면 최선을 다해 팀 프로젝트에 참여하고 도움이 될 수 있도록 노력하겠습니다. 감사합니다.
